Componentes requeridos:

    1. Ventana flotante:
        - Generar constante que almance el titulo de la ventana, el texto que se mostrara en la ventana, el texto debera mostrarse con una animacion de escritura,
        - Debera tener un boton de cerrar (dependiendo de que se este mostrando en la pantalla se podra cerrar o no se podra cerrar)
    
    2. Notificacion (Draggable):
        - Generar constante que almacene nombre de usuario que se mostrara en la notificacion, que guarde el texto y se trunque dependiende de si pase de la cantidad de caracteres maxima para que se no desborde de la pantalla (MessageTruncated), agregar un timestap de la hora en la que envia el mensaje
        - Cuando se clickee se debera mostrar el nombre de la persona y el texto completo de lo que se envia en una burburja de chat
    
    3. Burbujas de chat:
        - Se necesitara una burbuja de chat con pico (como en wsp)
        - Si la burbuja es de un mensaje recibido se debera mostrar con el pico en la parte izquierda, si no, se mostrara con el pico a la derecha
    
    4. Chat:
        - Se necesitara una chat que muestre las burbujas de chat 
        - Este chat se abrira con cada notificacion que se muestre y de primer mensaje debera mostrar el mensaje de la notificacion.
        - Se necesitara un input (con texto prestablecido) y un boton de enviar

    5. Ventanas de dialogo (Terminal estilo kitty):
        - Se necesitara una ventana de dialogo con el personaje Sailor Moon fumando que se comunicara con el usuario que interactua con la experiencia, dandole consejos, una que otra palabra de motivacion
        - Los textos de comunicacion de guardaran en una constante
        - Se necesitara un video de Sailor moon fumando en bucle 
        - Cuando aparezca el texto de Sailor Moon se mostrara el texto en una animacion de escritura
        - Las ventanas de dialogo apareceran dependiendo del momento de la escena en la que estemos
